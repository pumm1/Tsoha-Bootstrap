# Tietokantasovelluksen esittelysivu

## Työn aihe

Työn aiheenani on online keittokirja, jonne käyttäjät voivat lisätä omia reseptejään ja selata muiden resetepjä. Reseptien selaaminen ei vaadi käyttäjätunnuksia, mutta niiden lisääminen (kuten myös muokkaaminen ja poistaminen, kunhan käyttäjä on sama, joka myös sen sinne lisäsi!) vaatii rekisteröitymisen ja sisäänkirjautumisen.
